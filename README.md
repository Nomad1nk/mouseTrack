# ハンドジェスチャー認識システム - Hand Gesture Recognition System

Pythonを使用した**ハンドジェスチャー認識** + **バーチャルマウス制御**システム。

## 特徴

### 1. 🖱️ **バーチャルマウス (Virtual Mouse)** - NEW!

- ✅ 手でコンピュータのマウスを操作
- ✅ カーソルの移動、クリック、ドラッグ
- ✅ 実際のマウスの全機能


## インストール手順

### 1. 必須要件

- Python 3.8以上
- ウェブカメラ
- ブラウザ (Chrome, Firefox, Edgeなど)

### 2. ダウンロード

```bash
git clone <repository-url>
cd handgesture
```

### 3. インストール

```bash
pip install -r requirements.txt
```

### 4. 実行

#### 🖱️ バーチャルマウス

```bash
python virtual_mouse.py
python eye_mouse.py
```


## 🖱️ バーチャルマウスのジェスチャー - シンプルなシステム！

| ジェスチャー | アクション | 説明 |
| --- | --- | --- |
| ☝️ **1本指 (人差し指)** | **カーソル** | **ファイルを探す** |
| ✌️ **2本指 (人差し指+中指)** | **左クリック** | **ファイルを選択** |
| 🎯 **3本指 (☝️+✌️+💍)** | **ドラッグ＆ドロップ** | **ファイルをドラッグ＆移動** |
| 👍 **親指アップ** | **ダブルクリック** | **ファイルを開く** |
| 🖐️ **5本指 (パー)** | **停止** | **停止 / キャンセル** |

### 🎯 ドラッグ＆ドロップ - 簡単！

1. **☝️ 1本指** → ファイルへ移動
2. **🎯 3本指を立てる** → 即座にドラッグ開始！
3. **🎯 3本指を維持** → ファイルをドラッグ
4. **☝️ 1本指に戻す** → ファイルをドロップ！ ✅

**ワンステップでドラッグ！クリック不要！**


**'q' キーで終了**
